element selector : html 태그를 선택하는 선택자로, 같은 태그 모든 곳에 적용된다는 단점이 있음


id selector : 특정 id를 가진 부분에만 적용

id는 html element를 고유하게 지칭하는 것 & 하나의 html문서에 id 중복 X


class selector : 같은 css를 적용하고 싶은 부분들s을 하나로 편성

바로 위 라인과 같은 CSS적용 됨


group selector : 여러 개를 묶어서 하나의 css를 적용하는 개념
